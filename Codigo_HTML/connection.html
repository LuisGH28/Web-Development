Crea una Base de Datos (puede ser la de tu proyecto) en MySQL.

Realiza las siguientes actividades:

  • Crea un archivo en PHP que permita la conexión a la Base de datos creada.
  • Genera un archivo que CIERRE la conexión.

En ambos casos, deberás de indicar por medio de un texto que las acciones indicadas se realizaron satisfactoriamente.

Nota: Crea el archivo de conexión de tal forma que los datos puedan ser distintos, esto es, cualquier usuario con su respectiva contraseña conectarse a cualquier base de datos.
